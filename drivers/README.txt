# ChromeDriver binaries go here.
#
# macOS:
#   - Download from: https://chromedriver.chromium.org/downloads
#   - Rename to: chromedriver_mac
#   - Run: chmod +x chromedriver_mac
#   - Run: xattr -d com.apple.quarantine chromedriver_mac
#
# Windows:
#   - Download from: https://chromedriver.chromium.org/downloads
#   - Rename to: chromedriver_win.exe
#
# OR: Leave this folder empty and the bot will use Selenium Manager
# to auto-detect and download the correct ChromeDriver version.
